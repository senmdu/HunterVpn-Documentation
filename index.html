HunterVPN API Documentation

1. Fetch Available Server List

Endpoint: https://209.250.225.12:3000/fetchServers

This API endpoint allows users to fetch a list of available servers.

Request

Method: POST

Headers:

x-api-key: required
x-country-code: UAE (optional)
x-device-name: iPhone 11 Pro Max (optional)
x-os: ios (optional)

Body: application/json

{
    "userId": "sentest155fd-e2bd-40d6-a139-ee44637bb0092"
}

Response

{
    "status": true,
    "result": [
        {
            "id": "HUNT-LON-001",
            "name": "LON-VPN01",
            "city": "London",
            "country": "United Kingdom",
            "countryCode": "GB",
            "ip": "209.250.225.12",
            "vpnKey": "PLACEHOLDER_VPN_KEY"
        }
    ]
}

2. Connect to VPN Server

Endpoint: {ip}/connect

This API endpoint allows users to connect to a VPN server using the IP fetched from the server list. Use the vpnKey from the previously fetched server details.

Request

Method: POST

Headers:

x-api-key: required
x-country-code: UAE (optional)
x-device-name: iPhone 11 Pro Max (optional)
x-os: ios (optional)

Body: application/json

{
    "userId": "sentest155fd-e2bd-40d6-a139-ee44637bb0092",
    "publicKey": "wgxBC54ac/H3TSts6E064WADRngyyCnXhmULk7g6NW4=",
    "vpnKey": "PLACEHOLDER_VPN_KEY"
}

Response

{
    "status": true,
    "result": "VPN Server Connected Successfully",
    "vpn": {

        "dns": "1.1.1.1, 8.8.8.8",
        "addresses": "10.0.0.11/24"
    },
